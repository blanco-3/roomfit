# 다중 사진 촬영 기준 (크기 측정 정확도용)

## 권장 장수
- 최소 4장, 권장 6장, 최대 10장

## 필수 앵글
1. 출입문 근처에서 방 전체 전경 1장
2. 반대편 코너에서 전경 1장
3. 창문/문틀이 보이는 샷 1장
4. 바닥-벽 경계(모서리) 강조 1장

## 품질 기준
- 조명 충분 (역광 피하기)
- 흔들림 없는 촬영
- 0.5x 또는 광각 사용 권장
- 기준 물체 포함 권장 (문/침대/A4)

## 금지 조건
- 과도한 필터/보정
- 가구를 가리는 물체 과다
- 극단적 어안 왜곡

## 분석 파이프라인(개발 단계)
1. 업로드 검증(장수/해상도/포맷)
2. 세그멘테이션(벽/바닥/문/창문)
3. 멀티뷰 기반 깊이 추정
4. 스케일 보정(기준 물체 + priors)
5. 배치 가능 영역 계산
