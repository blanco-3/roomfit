# PRD v0.1 — AI 방 꾸미기 플랫폼 (MVP)

## 1. 문제정의
사용자는 방 사진을 찍어도 실제로 어떤 가구가 들어가는지, 통로가 확보되는지, 무드가 맞는지 판단하기 어렵다. 결과적으로 구매 결정이 느리고 반품이 잦다.

## 2. 제품 목표
- 입력: 방 사진/방 치수/예산/무드/용도
- 출력: 실제 배치 가능한 가구 추천 + 구매 링크
- 핵심 가치: **공간 적합성 + 취향 적합성 + 구매 가능성**

## 3. 타겟 사용자
- 원룸/오피스텔 거주 20~39세
- 이사/자취 시작, 홈오피스 구성 니즈
- 가구 구매 경험은 있으나 공간 설계 경험은 낮음

## 4. 핵심 기능 (MVP)
1) 방 프로파일 생성
- 방 크기(폭/길이/높이), 문/창문 위치(추후)
- 사진 기반 추정(후속), 현재는 입력+기본 휴리스틱

2) 카탈로그 정규화
- 가구 치수(width/depth/height), 가격, 스타일, 구매링크

3) 추천 엔진
- Hard filter: 예산, 치수, 통로 확보
- Soft rank: 무드/용도 일치도, 가격 효율

4) 추천 결과 UI
- 추천 카드, 선택 이유, 합계 가격

## 5. 성공 지표 (초기)
- 추천 클릭률(CTR) > 25%
- 추천 결과 저장률 > 15%
- 구매 링크 이동률 > 10%

## 6. 비기능 요구사항
- 추천 응답 p95 < 2.0초 (캐시 포함)
- 카탈로그 10k SKU까지 확장 가능한 구조
- 추천 근거 설명 가능성(Explainability)

## 7. 향후 로드맵
- 사진 세그멘테이션 + depth 기반 치수 추정
- 2D/3D 배치 시뮬레이터
- 멀티 쇼핑몰 동기화/API 제휴
- 개인화 추천(협업 필터링 + LTV 기반)
